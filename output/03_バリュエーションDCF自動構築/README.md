## 演習の概要

- **所要時間**: 約45分
- **難易度**: 中上級
- **使用ツール**: Claude Code Web（Sonnet 4.6）＋ GitHub Web
- **業界バリエーション**: 小売（RT・小売事業DCF）
- **到達率**: 約50%（Agent◎・軽量版）
- **データ型**: 型B派生（表形式→モデル構築）

### 業務の課題（要約）
企業価値算定のDCFモデル雛形が整備されておらず、案件ごとにゼロベースで作成。1モデルの構築に8〜16時間。PL/BS/CF・WACC・感度分析の標準化が属人的で、再利用が進まない。

### ありたい姿（要約）
過去の事業計画Excelから抽出されたモデル構築ロジックに基づき、パラメータ入力だけで5〜10年モデルが自動生成。WACC/IRR/NPV算出と感度分析テーブルがワンステップで出力。構築時間8〜16時間→2時間以内に短縮。

## ⚠️ 本演習の重要な前提（到達率50%の軽量版）

本演習は**3年軽量版**であり、以下は**対応不可・手動補正前提**です:

| 対応可（機械判定50%） | 対応不可（人の判断で補完） |
|---|---|
| 3年明示期間のPL構造 | 5〜10年長期計画 |
| FCF算出（NOPAT起点・簡易運転資本） | BS連動の精緻計算（貸借一致） |
| WACC算出（標準CAPM） | 複雑な運転資本計算（季節変動・回転日数の年次変動） |
| ターミナルバリュー計算（Gordon） | 税効果会計・繰延税金・のれん償却・減損テスト |
| 感度分析2軸×4シナリオ | 3パラメータ×9シナリオ感度分析 |

> 💡 **実務規模での注意**: 本演習は3年軽量版ですが、実務ではClaude Codeデスクトップ版（ローカル実行）を使えば5〜10年の長期モデル、3軸9シナリオ感度分析、BS連動も処理可能です。Web版は出力制限があるため、研修では3年軽量版で流れを体験することに集中します。

## 参加者の進め方

1. **準備（15分）**: 手順書§2〜§3に従いGitHubアカウント準備・リポジトリ作成・過去3件（A/B/C）をアップロード
2. **Step1（10分）**: Claude Code Webに過去3件を読み込ませ、CLAUDE.mdに3年軽量版DCFモデル構築ロジックを定義
3. **Step2（10分）**: 新規案件パラメータシート（アルビレオ小売）を追加アップロードし、CLAUDE.mdに従って1ファイル統合MD（3セクション）を生成
4. **振り返り（10分）**: 手順書§7のチェックポイントと模範出力（ファイル6・7）で到達度確認
